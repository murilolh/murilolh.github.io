About

I’m a Brazilian software developer, born and raised in Fortaleza - Ceará, and I’ve been working full-time in the field since 2010. Currently living in Toronto, ON, Canada.

In 2007, I started a Computer Science course at the Federal University of Ceará - UFC. Until my graduation in 2011, I learned the mathematical, logical and computational bases on this very theory-focused course. I’ve also had the opportunity to gain my first job experience as an IT Intern working in one of the department’s laboratories.

In 2009, I felt the need to start getting experience as a Software Developer, which was my aimed profession since before high school. I started searching for companies with trainee programs and applied for one at a local company called Xseed. It was a great starter experience because I spent the first months in a training period learning Java, Javascript, SQL and basic System’s analysis. After this period, I started working on migration projects from legacy COBOL/LINCII or Natural/ADABAS to Java / Oracle. We used a Java application to parse legacy code to Java, which I helped to expand and maintain. At the end of 2010, my classes’ schedule allowed me to work full-time and I was hired as a Junior Software Developer.

After my graduation in 2011, I started working at Cafaz, a health insurance company. They had bought an application to control all areas of the company, developed in NodeJS. This application allows new code to be added by the clients, having its own version control and editor, and they needed someone to develop and maintain this customized code. It was a chance to get more knowledge of JavaScript, Oracle and SOAP Web Services as well as Requirements Elicitation, since I was in constant contact with both internal and external users. Sometimes they also required me to build Mobile Applications and PHP+Apache+MySQL systems when it wasn’t possible/needed to be coded within the main system. In 2016, a few years after my departure, they contacted me to work remotely as a part-time developer, a position that I held until 2018.

In 2013, I was selected for an IT Analyst position at Dataprev, a federal IT company in Brazil. At this company I was able to consolidate some knowledge while working on web systems with JSF, EJB and JPA/HIbernate. It was awesome to participate in a well-established and agile software development cycle, working from project’s planning until its deployment in production. There I was able to work on Testing activities, like participating in integration tests with both internal and external teams and developing unit, integration, automated and performance tests. Dataprev’s environment also allowed me to learn a lot through some challenges throughout the years, like working on large Spring Batch projects and being a substitute project manager for some time. Later on, it was great being able to migrate to a more service-oriented architecture using REST APIs.

I’ve always wanted to live and work abroad, and Canada was my country of choice, as it would be both a great life and professional experience. In 2017, I started a process to be a Canadian Permanent Resident, which was concluded in 2018, when I arrived in Toronto.

In early 2019, I started working at Veeva, developing features for a cloud based system related to Regulatory Submissions to Health Authorities. There I was able to improve my JAVA/Spring backend development capabilities, work with React at the frontend, engage in technical product support for customer facing issues as well as working with QA and PMs to drive new product features from planning to production.

For more information about my professional career, you can refer to my LinkedIn and GitHub pages.
